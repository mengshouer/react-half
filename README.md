## 🔨 使用

```bash
$ git clone https://github.com/mengshouer/react-half.git
$ cd react-half
$ git checkout ql
$ cp .env.example .env
$ pnpm i
$ pnpm dev
```

> 首次使用请自行修改 .env 中的配置
